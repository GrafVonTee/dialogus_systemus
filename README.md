# Диалоговая система

Первая попытка организовать **диалоговую систему** для персонажей и событий.

Включает в себя:
- Дневник похождений: *парсер и редактор в разработке*
- Дневник терминов: *парсер и редактор в разработке*
- Систему сообщений: *даже не начиналось*
- Редактор: *пока даже не начинался*
